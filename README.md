# 🎨 StylishCalc

A beautiful, animated, big-font macOS calculator by **Gityus13**

## Install via Homebrew
```bash
brew tap Gityus13/tap
brew install --cask stylish-calc
```

## Features
- 🔢 Giant animated display with gradient font
- 💜 Purple glow operator buttons  
- 🌌 Deep space gradient background
- ⚡ Spring-bounce animations on every tap

## First Launch

When you first open StylishCalc, macOS may show a security warning because the app isn't notarized with Apple (requires $99/year). 

**To open:** Right-click the app and select "Open", then click "Open" in the dialog. You only need to do this once!
